# DEM 資料檔案

此目錄包含 DEM（數值高程模型）資料檔案。

## 檔案說明

- `dem_20m_hualien.tif` - 花蓮地區 20m 解析度 DEM
  - 檔案大小：107.60 MB
  - 來源：內政部國土測繪中心
  - 解析度：20m x 20m
  - 坐標系統：TWD97

## 下載方式

由於檔案超過 GitHub 100MB 限制，請透過以下方式取得：

1. **Google Drive**：[下載連結]（請放置實際的 Google Drive 連結）
2. **內政部平台**：從國土測繪中心官方平台下載
3. **課程平台**：從課程指定位置下載

## 使用說明

下載後請將 `dem_20m_hualien.tif` 檔案放置於此目錄中，然後執行 ARIA_v2.ipynb 分析。

注意：確保檔案名稱完全符合 `dem_20m_hualien.tif`，否則程式碼可能無法正確讀取。
